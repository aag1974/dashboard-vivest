Dashboard Master - Header Overlay

Sistema otimizado onde análises SPSS sobrepõem toda a interface com header próprio, mantendo apenas sidebar para navegação.

🚀 Arquitetura v2.0

  • ✅ Header SPSS fixo sobrepõe toda a tela
  • ✅ Filtros compactos em linha única no header
  • ✅ Máximo aproveitamento de espaço vertical
  • ✅ Menu lateral mínimo sempre acessível
  • ✅ Comunicação integrada via postMessage

Como usar:

  1. Gere análises com criar_dashboard_v2.py
  2. Adicione no menu usando este gerenciador
  3. Navegue - cada análise ocupa toda a tela
  4. Use sidebar para trocar entre análises
